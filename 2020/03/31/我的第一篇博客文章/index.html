我的第一篇博客文章

hexo 配置

准备环境

[node] (https://nodejs.org/en)
npm 现在安装完node后有自带但是npm网站在国外，可能比较慢，可以通过安装淘宝镜像来改善：

npm install -g cnpm –registry=https://registry.npm.taobao.org;

安装hexo (是基于node的)

npm install -g hexo-cli
通过 hexo -v 验证

安装 git
网址：（msysgit.github.io）

git config –global user.name “xx” [xx为任意名]
git config –global user.email “xxxxx” [xxxxx设置邮箱]

搭建git服务器（远程仓库） ：统一的托管网站（https://github.com/）
为了 在本地 和远程仓库之间进行 免密钥登录，可以配置ssh

在任意地方新建文件夹（在搭建过程中有错误可以直接 干掉文件夹）
初始化博客:

hexo init

启动博客：

hexo s

创建一篇文章

hexo n “我的第一篇测试文章”（生成的文章在source/_posts）

然后开始编写内容……

hexo clean (清理)
hexo g (生成)
hexo s (启动)

将hexo部署到github上去

登录github新建仓库（用户部署个人博客的仓库命名必须符合特定规范）
账号名称.github.io
安装部署插件

npm install hexo-deployer-git –save

设置_config.yml文件
deploy:
type: git
repo: https://github.com/xxxx/xxx.github.io.git

部署命令

hexo d (输入github账号密码)

访问github仓库
